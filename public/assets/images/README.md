# Images publiques BÂTIZEN CI

Placez vos images dans ce dossier avec les noms suivants pour que l'application les utilise automatiquement.

## Noms de fichiers nécessaires

- `logo.png`
  - usage : logo principal, splash, header
  - taille recommandée : 512×512 px
  - format : PNG (ou SVG si possible)

- `hero-villa.jpg`
  - usage : image d'accueil / onboarding
  - taille recommandée : 1600×900 px
  - format : JPG ou WebP

- `project-villa-abidjan.jpg`
  - usage : image de projet sur le dashboard et les cartes
  - taille recommandée : 1200×800 px
  - format : JPG ou WebP

- `project-duplex-yamoussoukro.jpg`
  - usage : deuxième image de projet
  - taille recommandée : 1200×800 px
  - format : JPG ou WebP

- `onboarding-site.jpg`
  - usage : illustration d'onboarding si nécessaire
  - taille recommandée : 1600×900 px
  - format : JPG ou WebP

## Comment lier vos images

1. Copiez vos fichiers dans `public/assets/images`.
2. Renommez-les avec les noms ci-dessus.
3. Rechargez l'application dans le navigateur.

## Remplacement recommandé pour vos fichiers

- Votre logo : `logo.png`
- Maison moderne principale : `hero-villa.jpg`
- Première image de projet : `project-villa-abidjan.jpg`
- Seconde image de projet : `project-duplex-yamoussoukro.jpg`

Si vous souhaitez que je renomme les fichiers du code pour utiliser d'autres noms, dites-moi les noms exacts que vous voulez utiliser.
