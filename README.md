# BÂTIZEN CI

Application web fullstack Next.js + PostgreSQL/Drizzle pour simulation, devis, suivi chantier, messages, profil, services Google/Firebase et dashboard premium.

## Lancer

- `npm run dev`
- `npm run build`

## Structure principale

- `src/app` : routes App Router
- `src/components` : UI, layout, cards
- `src/services` : données BÂTIZEN et Google/Firebase
- `src/db` : Drizzle ORM PostgreSQL
- `public/assets/images` : logo et images premium
- `docs` : bible projet officielle
